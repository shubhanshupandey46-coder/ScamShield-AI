 # 🛡 ScamShield AI

> **AI-Powered Scam Detection using Google Gemini**

ScamShield AI is an intelligent web application that detects phishing and scam messages using Google's Gemini AI. Instead of relying on simple keyword matching, it performs contextual analysis to evaluate suspicious SMS, emails, WhatsApp messages, and social media texts, then provides an explainable risk score along with safety recommendations.

# ✨ Features

- 🤖 Google Gemini AI-powered contextual analysis
- 🛡 Detects phishing and scam attempts
- 📊 Explainable AI risk score (0–100)
- 🚨 Scam / Suspicious / Safe verdict
- 📋 Detailed reasoning behind each prediction
- 💡 Personalized safety recommendations
- 🌌 Modern cybersecurity-inspired UI
- 🎨 Glassmorphism interface
- ⚡ Animated circular risk gauge
- 📱 Responsive design
- ☁️ Fully deployed on Render

# 🚀 Installation

```bash
git clone https://github.com/shubhanshupandey46-coder/ScamShield-AI.git

cd ScamShield-AI
```

Backend

```bash
cd backend
npm install
npm start
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
GEMINI_API_KEY=YOUR_API_KEY
```
