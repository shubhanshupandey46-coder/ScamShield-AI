# 🛡 ScamShield AI

An AI-powered scam detection assistant built during an MLH Hackathon.

## 🚀 Features

- Detect phishing messages
- Analyze suspicious SMS
- AI-powered risk score
- Scam explanation
- Safety recommendations

## Tech Stack

- React
- Node.js
- Express
- Google Gemini API

## Future Improvements

- URL Scanner
- QR Code Detection
- Voice Scam Detection
- Browser Extension
- OCR Image Analysis
