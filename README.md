# рҹӣЎ ScamShield AI

> **AI-Powered Scam Detection using Google Gemini**

ScamShield AI is an intelligent web application that detects phishing and scam messages using Google's Gemini AI. Instead of relying on simple keyword matching, it performs contextual analysis to evaluate suspicious SMS, emails, WhatsApp messages, and social media texts, then provides an explainable risk score along with safety recommendations.
